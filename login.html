Como usar

1. Crie um usuário — defina login, senha e o nome do arquivo .html para onde ele será redirecionado.

2. Coloque os arquivos HTML na mesma pasta que este login.html.

3. Os dados ficam salvos no localStorage do navegador. Use Exportar JSON para fazer backup e Importar JSON para restaurar.

4. O usuário master é o administrador principal e não pode ser removido.
